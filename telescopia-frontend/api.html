Telescopia API


O telescopia disponibiliza uma API Rest, com Cross-Origin Resource Sharing (CORS) ativado, utilizando o protocolo HTTP para que você mesmo possa manipular os dados em formato JSON, sem interferência deste website.
GET    https://186.194.221.240/

Mostra a "introdução" da API, como versão e nome.

{
    "title": "Telescopia API",
    "version": "0.0.1"
}

Atenção: Datas são aceitas apenas no formato Y-m-d.


Notícias

O sufixo /articles/ na URL representa a seção de notícias da API.

{
    "status": "success",
    "message": [
        {
            "_id": "5de7d159e408f077d9bcc22e",
            "url": "https://www.noticias.com.br/noticia/1",
            "title": "Notícia 1",
            "body": "Corpo da notícia 1",
            "date": "2019-11-08T19:32:40.000Z",
            "portal_name": "Portal Notícias",
            "section": "economia",
            "comments": [],
            "__v": 0
        }
    ]
}

Exemplo de retorno dado pela API nesta seção.

GET    https://186.194.221.240/articles/

Retorna todas as notícias já cadastradas no banco de dados.

GET    https://186.194.221.240/articles/getArticle/:id

Retorna uma notícia, onde :id deve ser subtituído pelo id da notícia.

GET    https://186.194.221.240/articles/getArticlesByDate/:date

Retorna todas notícias publicas na data passada em :date.

GET    https://186.194.221.240/articles/getArticlesAfter/:date

Retorna todas notícias que foram publicadas após a data especificada em :date.

GET    https://186.194.221.240/articles/getArticlesBefore/:date

Retorna todas notícias que foram publicadas até a data especificada em :date.

GET    https://186.194.221.240/articles/getArticlesByDateRange/:dateLow/:dateHigh

Retorna todas notícias que foram publicadas entre as datas especificadas em :dateLow e :dateHigh.

GET    https://186.194.221.240/articles/getNumberOfArticles/

Retorna o número de notícias cadastradas no banco de dados.


Ações

O sufixo /stocks/ na URL dá acesso à seção de ações da API.

{
    "status": "success",
    "message": {
        "_id": "5de7d2fae408f077d9bcc3b7",
        "name": "OIBR3",
        "stats": [
            {
                    "_id": "5de7d2fae408f077d9bcc3b8",
                    "url": "https://www.infomoney.com.br/mercados/acao-do-bb-sobe-mais-de-1-apos-anuncio-de-oferta-de-acoes-br-distribuidora-avanca-3-com-recomendacao/",
                    "date": "2019-10-03T10:32:07.000Z"
            },
            {
                    "_id": "5de7d622e408f077d9bccb81",
                    "url": "https://www.infomoney.com.br/politica/petista-e-acusado-de-oferecer-propina-a-conselheiro-da-anatel-em-nome-da-oi/",
                    "date": "2013-08-26T06:36:00.000Z"
            }
        ]
    }
}

Exemplo de retorno da API nesta seção.

GET    https://186.194.221.240/stocks/

Retorna todas ações cadastradas no banco de dados.

GET    https://186.194.221.240/stocks/getStock/:stockName

Retorna uma ação pelo seu nome especificado em :stockName.

GET    https://186.194.221.240/stocks/getNumberOfStocks/

Retorna o número de ações cadastradas no banco de dados.


Histórico de Ações

O sufixo /stocks/history/ provê o histórico de ações desde o ano 2000. Neste histórico estão incluídos o valor de abertura, de fechamento, valor mínimo, máximo e a variação, para uma data especificada.

{
    "status": "success"
    "message": {
        "name": "OIBR3",
        "historic": [
        ]
    }
}

Exemplo de retorno desta seção.

GET    https://186.194.221.240/stocks/history/

Retorna todas ações e seus respectivos históricos.

GET    https://186.194.221.240/stocks/history/getStockHistoryByName/:stockName/

Retorna todo o histórico de uma ação, cujo nome deve ser posto em :stockName.

GET    https://186.194.221.240/stocks/history/getStockHistoryByDate/:date/

Retorna o histórico de todas ações para uma determinada data especificada em :date.
